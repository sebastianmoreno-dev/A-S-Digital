¿Qué es?

La "memoria" y el cerebro de tu sitio web

Un sitio web básico solo muestra información — como un folleto digital. Pero si necesitas que haga algo — guardar datos, procesar pedidos, manejar cuentas de usuario, enviar correos automáticos — eso requiere un backend.

El backend es el servidor y la base de datos que trabajan detrás de tu sitio, invisibles para el usuario pero esenciales para que todo funcione. Lo construimos con Python y Flask: rápido, seguro y fácil de mantener.

Si estás empezando desde cero, probablemente lo que necesitas es el Sitio Web Completo que ya incluye diseño y backend juntos.
